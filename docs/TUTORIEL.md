# Tutoriel d'utilisation - PlombiCRM

## 1. Prise en main rapide
1. Lancez le backend (voir `docs/BACKEND.md`).
2. Ouvrez `http://localhost:3000` dans votre navigateur.
3. Utilisez le menu de gauche pour naviguer entre les modules.
4. Vérifiez que la date et les indicateurs s'affichent dans le tableau de bord.

## 2. Gestion des clients
- Allez dans **Clients** puis remplissez le formulaire "Nouveau client".
- La base clients se met à jour automatiquement.
- Utilisez la barre de recherche globale pour filtrer rapidement.

## 3. Création d'un devis automatisé
1. Ouvrez **Devis**.
2. Sélectionnez un client, un service, des matériaux et le nombre d'heures.
3. Cliquez sur **Générer le devis**.
4. Utilisez **Envoyer par email** pour ouvrir le message prérempli.
5. Cliquez sur **Confirmer réception** quand l'accusé de réception arrive.

## 4. Suivi des projets
- Allez dans **Suivi projets**.
- Ajustez l'avancement via le curseur.
- Le statut bascule en "Terminé" quand l'avancement atteint 100 %.

## 5. Rapports et analyses
- Consultez le taux de conversion, les tendances de vente et les services demandés.
- Vérifiez la satisfaction client pour piloter la relation.

## 6. Intégrations
- Activez ou désactivez les intégrations disponibles (comptabilité, planning, etc.).
- Utilisez ces réglages comme base pour vos futures synchronisations.

## 7. Réinitialiser les données
Cliquez sur **Réinitialiser les données** dans le tableau de bord pour revenir à la base d'exemple.
